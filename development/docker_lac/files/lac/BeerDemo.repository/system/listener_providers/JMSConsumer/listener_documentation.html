Live API Creator uses JMSConsumer listeners to subscribe to messages in queues in JMS-compliant providers, such as ActiveMQ and IBM MQ. They use a JMSConsumer connection to connect to the exchange. When the listener receives a message along the route endpoint defined by the JMS listener parameters, Live API Creator executes the listener JavaScript code.

For more information about how to create JMSConsumer listeners, see JMS Listeners.
