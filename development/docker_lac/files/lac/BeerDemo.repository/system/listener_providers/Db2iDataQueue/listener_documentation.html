Live API Creator uses Db2iDataQueue listeners to subscribe to messages from data queues that reside on IBM Db2 for i servers. Db2iDataQueue listeners use Db2iDataQueue connections to connect to these data queues. When the listener receives a message from the data queue, Live API Creator executes the listener JavaScript code.

For more information about how to create a Db2iDataQueue listener, see Db2iDataQueue Listeners.
