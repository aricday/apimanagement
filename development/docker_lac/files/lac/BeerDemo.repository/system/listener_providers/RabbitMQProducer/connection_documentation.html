Connection

Connections create and ready a CamelContext that provide access to among many other camel specific items - components, type converters, and endpoints. Connections could also have parameters depending on its connection type and consequently the component it uses. An active Connection JavaScript object can be retrieved by calling getConnection method in Sysutility object.

For more information about this method, see Sysutility.getConnection()
