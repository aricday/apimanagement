Shutdown Listeners

A shutdown listener is a piece of code that gets executed any time a Live API Creator server is shut down.

More specifically, it gets executed any time the WAR file for Live API Creator is stopped or restarted.

You can create a shutdown listener and have it execute any JavaScript code. Typical examples might include:

  • writing out some results to a file before the service stops
  • notifying an outside service that this Live API Creator is coming down
  • cleaning up anything that was started by a startup listener
