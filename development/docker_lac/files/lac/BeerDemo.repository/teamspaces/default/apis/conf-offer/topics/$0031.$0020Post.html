As a Marketing Manager, I want to enable our Partners to send us Conference Offers to participate in their events.

Offers provide Options for Talks (Speaking Opportunities), and Exhibits.  These include Costs.

Acceptance Criteria:

  1. Partner can post with this information
  2. Post request data is audited, for admin tracking and error recovery.
  3. Conference Offer data is stored in a local database
  4. A Web App, for users to mark Talks/Exhibits as Used, and approve offers
